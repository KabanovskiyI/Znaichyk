# Розробка інтерактивного сайту для розвитку дітей дошкільного віку

Кваліфікаційна робота на підтвердження ступеня фахового молодшого
бакалавра ([ВСП «ППФК НТУ «ХПІ»](http://polytechnic.poltava.ua)). Керівник
роботи – **Олійник Владислав Васильович**.

## Завдання до дипломної роботи

розроблений програмний продукт має являти собою веб
сайт з інтерактивними завданнями для дітей. Він має надавати всю необхідну
функціональність: інтуїтивно зрозумілий інтерфейс, зрозумілі завдання,
голосовий супровід в завданнях, тощо. Для адміністратора окремо створено
базу даних, де можна додавати нові завдання. Розробку виконати мовою
програмування JavaScript використовуючи React.js та Node.js. Для
графічного зображення використати Chakra UI.

## Використані технології

JavaScript, React, NestJS, PostgreSQL
